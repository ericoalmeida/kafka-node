## gateway

Simula o gateway (API principal) que irá receber as requisições.

### Bibliotecas

- kafkajs
- express

### Projeto

Para iniciar o projeto execute os comandos abaixo:

- Instale as dependências

```shell
yarn
```
ou
```shell
npm install
```

- Execute o projeto

```shell
yarn dev
```
ou
```shell
npm run dev
```
