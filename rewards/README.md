## Rewards

Simula o service (Engine of rewards) que irá processar os bonus e disponibiliza-lo ao cliente.

### Bibliotecas

- kafkajs

### Projeto

Para iniciar o projeto execute os comandos abaixo:

- Instale as dependências

```shell
yarn
```
ou
```shell
npm install
```

- Execute o projeto

```shell
yarn dev
```
ou
```shell
npm run dev
```
